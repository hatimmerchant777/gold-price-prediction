About the Project

Gold Price Prediction Using Machine Learning

This project implements various machine learning algorithms to predict gold prices:

  • Linear Regression: Uses linear relationship between features to predict prices
  • Logistic Regression: Predicts price trends (higher/lower than median)
  • Decision Tree: Creates a tree-based model for price prediction
  • SVM: Uses Support Vector Machine for regression
  • Naive Bayes: Probabilistic predictions based on feature distribution
How to Use
  1. Enter the future date (Year, Month, Day)
  2. Select your preferred prediction model
  3. Click "Predict" to get the estimated gold price
